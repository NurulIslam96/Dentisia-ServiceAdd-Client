## Features

- Mobile Compatibility
- Fullscreen mode
- Cross platform
- React Hot Toasts
- Firebase Authentication
- Custom Error 404 Page
- User Friendly UI
- Custom API
- Image Zoom on Click
- Review Page with Edit and Delete Function
- Reviews sorted by Timestamp
- Animation on Scroll


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
